Variáveis em Python

1. O que são Variáveis?

Uma variável é um espaço na memória que armazena dados que podem ser alterados durante a execução do programa.

2. Tipos de Variáveis

  • Inteiros (int): Números sem ponto decimal, como 5, 10, -3.
  • Números de Ponto Flutuante (float): Números com ponto decimal, como 3.14, -0.001.
  • Strings (str): Sequências de caracteres, como "Hello, World!".
  • Booleanos (bool): Valores que representam verdadeiro ou falso, True ou False.
  • Listas: Coleções ordenadas e mutáveis de itens, como [1, 2, 3] ou ['a', 'b', 'c'].
  • Tuplas: Coleções ordenadas e imutáveis, como (1, 2, 3).
  • Dicionários: Coleções de pares chave-valor, como {'nome': 'Alice', 'idade': 25}.

3. Declaração de Variáveis

nome = "Alice"
idade = 25
altura = 1.68
estudante = True

4. Nomenclatura de Variáveis

Regras:

  • Uma variável deve começar com uma letra (a-z, A-Z) ou um sublinhado (_).
  • O restante do nome pode conter letras, números (0-9) e sublinhados.
  • Não pode ser uma palavra reservada (como if, else, while, etc.).

Boas Práticas:

  • Use nomes descritivos para facilitar a compreensão do código.
  • Use letras minúsculas e sublinhados para separar palavras (ex.: idade_do_aluno).

5. Tipagem Dinâmica

Python é uma linguagem de tipagem dinâmica, o que significa que você não precisa declarar o tipo da variável. O tipo é inferido a partir do valor atribuído:

x = 10       # x é um inteiro
x = "Texto"  # Agora x é uma string

6. Operações com Variáveis

Você pode realizar várias operações com variáveis, como:

Aritméticas:

a = 10
b = 3
soma = a + b          # 13
diferenca = a - b     # 7
produto = a * b       # 30
divisao = a / b       # 3.333...
divisao_inteira = a // b  # 3
resto = a % b         # 1
exponenciacao = a ** b  # 1000

Concatenação de Strings:

saudacao = "Olá, " + nome  # "Olá, Alice"

7. Escopo de Variáveis

Variáveis Locais:

def minha_funcao():
    x = 10  # x é local
    print(x)

Variáveis Globais:

x = 20  # x é global

def outra_funcao():
    print(x)  # Acessa a variável global

8. Variáveis Mutáveis e Imutáveis

Mutáveis:

lista = [1, 2, 3]
lista.append(4)  # Agora a lista é [1, 2, 3, 4]

Imutáveis:

nome = "Alice"
nome[0] = "a"  # Isso causaria um erro, pois strings são imutáveis

9. Atribuição Múltipla

a, b, c = 1, 2, 3

10. Variáveis e Entrada do Usuário

nome = input("Qual é o seu nome? ")
idade = int(input("Qual é a sua idade? "))  # Converte a entrada para inteiro
